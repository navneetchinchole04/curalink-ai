# 🧠 CuraLink AI

CuraLink AI is a medical research assistant that helps users find and understand research papers using AI.

## 🚀 Features
- 🔍 Search research papers (OpenAlex + PubMed)
- 🤖 AI-generated summary using local LLM (Ollama)
- 📄 Top relevant papers with year
- ⚡ Fast and lightweight

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express
- APIs: OpenAlex, PubMed
- AI: Ollama (phi model)

## 📦 Setup

### 1. Clone repo
```bash
git clone https://github.com/navneetchinchole04/curalink-ai.git
cd curalink-ai
```

---

2. Backend

cd backend
npm install
node server.js

---

3. Run LLM

ollama run phi

---

4. Frontend

cd frontend
npm install
npm start

---

Search:

lung cancer treatment
